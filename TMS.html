Tenant Management System


TMS is an single sign-on portal users only have to enter one set of credentials to access all the web applications – via desktops, smartphones and tablets.
This greatly increases productivity while keeping data secure.

Project Span & Duties


6 Months
Competitive Analysis
Wireframing, Prototyping, UI Designing
UX UI Designer and Developer

Problem & Goals


User Usability, Avoid Mutiple Logins, Security, Simplifies username and password management, Reduces security risks for your customers, vendors, and partner entities. The goal is to improve user’s usability by allowing a single sign on policy and improving security.

Pain Points


Multiple logins for suite of products.
Keep a track of multiple usernames and passwords, compromising the security.
Inconvenient and unhealthy user experience.
