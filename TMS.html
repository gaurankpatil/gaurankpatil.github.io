Tenant Management System

Project Overview

TMS is an single sign-on portal where users only have to enter one set of credentials to access all the web applications – via desktops, smartphones and tablets. This greatly increases productivity while keeping data secure. The tenant is a framework within which system objects such as users and machine images are created. The tenant can be an organization, a department within an organization, or an individual.

Problem Statement

  • User usability
  • Avoid mutiple logins
  • Security
  • Simplifies username and password management
  • Reduces security risks for your users
  • Inconvenient user experience
