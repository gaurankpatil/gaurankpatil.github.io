Buyer/Supplier Portal


ERFX is an multipurpose buyer and supplier portal. ERFX is designed and developed to send out RFP's, RFI's and RFX's to vendors and collate their responses automatically. With advance tools for creating an RFX ie and combination of RFI,RFX and RFP. ERFX keeps expanding its suite of features to provide great user experience and security. Its designed and developed using adavance UI technologies and tools for enhanceing overall user experience. ERFX provides tab view which is simplified and easy to understand. Discover Supplier feature allows an RFX creator to search the suppliers which are catogrised according to each item sold by the suppliers. Checklist feature allows an comparatitive view of the all the suppliers associated with each RFX rasied. It also provides and Q&A view for any questions or queries raised by each suppliers for the particular RFX. An table view of the invited suppliers in the organization. A editable view of user profile with company details and profile picture option available.
