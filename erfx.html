ERFX

Project Overview

ERFX stands for Electronic Request For [x], where x can be Proposal (RFP), Quotation (RFQ), Information (RFI). ERFX is an multipurpose buyer and supplier portaldesigned and developed for buyers and suppliers for raising RFP’s and RFX’s events and collate their responses automatically in the system. combined with advance suite of features to provide great user experience and security for raising an RFX ie and combination of RFI,RFX and RFP's. ERFX provides simplified tabbed view architecture divided in multiple sections for ease of user experience while raising an RFX. Its a combination of multiple features encorporated in a single portal providing the best user experience, usability and accessability. This portal allows a clear view of the each and every stage an RFX flows throughout while mainting the confindentiality and security of each RFX raised.
The basic features included are as follows:
1. Discover supplier feature.
2. In built query portal.
3. Advanced profile section with complete user details and profile picture.
4. Compatible with all browsers and across all devices.
5. Built with advanced UX/UI technologies to get the most delightful user experience.
6. Responsive for every device and supports cross browser compatibility.
