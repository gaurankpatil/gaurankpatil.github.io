Query Management Portal


Query Management Portal is an new chat box feature incorporated in EMS. It is a form of synchronous communication carried out to provide quick resolutions for any queries raised related to its respective voucher during approval process. This is an powerful and effective communication tool that fosters a fascinating, authentic user experience. Query Management Portal has all the feature of an modern chat application ie. including attachment facility and providing total count of queries raised for a particular voucher etc.
